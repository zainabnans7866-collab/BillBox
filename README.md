# 🧾 BillBox AI

BillBox AI is an AI-powered warranty and bill management system that helps users securely store invoices, track warranty expiry dates, and manage purchases from one dashboard.

## 🚀 Features

- 🔐 Supabase Authentication (Login & Signup)
- 🛡 Protected Routes
- ➕ Add Bills
- ✏ Edit Bills
- ❌ Delete Bills
- 📄 View Bill Details
- 🤖 AI Bill Scanner using Groq Vision
- 📧 Email Notifications using Resend
- ☁ Bill Image Storage using Supabase Storage
- 📊 Analytics Dashboard
- 🎨 Responsive Modern UI
- 🌙 Dark Theme

## 🛠 Tech Stack

- React.js
- Vite
- Tailwind CSS
- Supabase
- Groq AI
- Groq Vision
- Resend
- React Router
- Lucide React

## 📂 Installation

```bash
git clone <repository-url>
cd BillBox
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file and add:

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
VITE_GROQ_API_KEY=YOUR_GROQ_API_KEY
VITE_RESEND_API_KEY=YOUR_RESEND_API_KEY
```
